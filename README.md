### 💎 V5 專業相容版：精選更新重點
本次 V5 更新專注於「全平台使用體驗」的極致修復，解決了多項跨平台技術痛點：

- **🛡️ Popup 攔截偵測**：主動偵測機制，若瀏覽器阻擋分頁彈出，將立即提示使用者開啟權限。
- **📱 行動端深度優化**：
  - **16px 縮放法則**：精準字體設定，防止 iPhone 點擊輸入框時畫面強制放大。
  - **智慧特效隱藏**：觸控裝置自動停用特效游標，解決手機端卡頓。
- **⚡ 強制焦點回歸**：使用 `blur()` 與 `focus()` 組合技術，優化批次開啟分頁後的視覺跳轉體驗。
- **🍅 阻尼物理進化**：精細調教番茄鐘的動態回饋，提供更自然的觸感。
