# 🚀 全能批次搜尋控制台 (Omni-Batch Search Console)

這是一個基於 HTML/CSS/JS 開發的高效率批次搜尋工具。從最初的雙字典查詢，進化到目前集成了搜尋引擎、字典與生產力工具的全能控制台。

### 🌟 V4 核心更新
- **全能搜尋**：新增 Google 與 Web 深度搜尋模式。
- **動畫優化**：更流暢的視覺反饋與 UI 交互。
- **番茄鐘升級**：改進阻尼物理效果，提供更自然的計時體驗。
