# 🚀 Omni-Search Hub V5 (Intelligent Evolution)
### 智能天氣感應 ✕ 12 國語言支持 ✕ 極致視覺入口

Omni-Search Hub V5 是一個結合了 **Vibe Coding** 美學與強大工具邏輯的發布入口網頁。此版本不僅是歷程的紀錄，更是一個具備「環境感知」能力的智能工作台。

---

## ✨ V5 版本關鍵更新

### 🌤️ 智能天氣建議系統 (Weather Advisory)
* **動態生活提案**：系統能根據即時天氣狀況（如：晴天、雨天、冷鋒）提供對應的搜尋建議或心情語錄。
* **氛圍同步**：UI 顏色與毛玻璃透明度會隨天氣狀態進行細微調整，讓數位工具更具「溫度」。

### 🌐 12 國語言矩陣 (Multi-Language Matrix)
為了打造真正的全球化工具，我們擴充了語言選單，並優化了長列表的滑動與顯示體驗：
* **語言列表**：英語、繁體中文、簡體中文、日語、韓語、西班牙語、葡萄牙語、越南語、孟加拉語、印地語、俄語、粵語。
* **UX 優化**：選單支援 `max-height` 滾動保護，確保在各種比例的行動裝置上皆不遮擋視線。

### 🖱️ 互動設計再進化
* **雙重指針系統**：同時保留系統原生滑鼠（確保操作精準）與「液態圓餅拖尾」特效（增加視覺張力）。
* **自適應特效控制**：
    * **Desktop**：啟用阻尼感磁吸動畫與游標追蹤。
    * **Mobile**：自動停用游標特效以節省效能，並優化觸控回饋。

## 🛠️ 技術細節與規格

| 功能 | 技術實作 | 效益 |
| :--- | :--- | :--- |
| **視覺風格** | Glassmorphism 2.0 | 使用 `backdrop-filter: blur(30px)` 打造高級透鏡質感。 |
| **主題管理** | CSS Variables + Data-Theme | 一鍵切換深淺色，所有顏色變數（包含陰影與漸層）自動適配。 |
| **跨平台相容** | Pointer-events Logic | 確保電腦版特效不干擾點擊，行動版不產生多餘殘影。 |
| **動畫曲線** | Cubic-Bezier (Spring) | 使用自定義貝茲曲線，讓卡片彈跳感更接近原生 App。 |

## 💡 使用與部署
1.  將 `index.html` 部署至 GitHub Pages 或任何 Web Server。
2.  確保 `V1` 至 `V5` 的檔案路徑與 `index.html` 處於同一層級。
3.  建議使用支援 `backdrop-filter` 的現代瀏覽器以獲得最佳視覺體驗。
