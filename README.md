### 💎 版本更新精華

#### **V5 專業相容版 (Latest)**
- **🛡️ Popup 攔截偵測**：主動提示使用者開啟彈出視窗權限，解決使用障礙。
- **📱 行動端適配**：防止 iOS 輸入框強制縮放 (16px 規則) 與智慧特效游標隱藏。
- **⚡ 焦點回歸**：優化批次開啟分頁後的視覺跳轉體驗。

#### **V4 全能控制台 (Major Update)**
- **🌤️ 生活功能集成**：**新增即時天氣查詢功能**，讓控制台更具生活實用性。
- **🔍 搜尋擴充**：全面集成 Google 與 Web 深度搜尋模式。
- **🎨 UI 響應**：動畫效果全面升級，提升操作流暢度。
